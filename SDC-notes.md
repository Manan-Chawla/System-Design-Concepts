# **__System Design Concept__**

## **System** 
A system is a group of an interdependent component linked together according to a plan to achieve specific objective. System means an organized relationship among functionality units or components.

## **Charaterstics of System**
1. Organization :- Also refer as the arrangement of component.
2. Interaction :- Defined as a way component operate with each other
3. Interdependence :- It means how component of system are dependent on each other.
4. Integration :- It is concered how a system component are connection together.
5. Central Objective :- Objective of a system has to be central so it can be real or stated.

## **Pros of System**
1. Improve efficency
2. Organized interaction 
3. Clear goal achievement

## **Cons of System**
1. Complexity
2. Potential for failure
3. Resistance to change
